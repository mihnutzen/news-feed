Exercise News Feed
==================

Single page app that displays latest 10 news from Yahoo pipes

Data URL: http://pipes.yahoo.com/pipes/pipe.run?_id=DqsF_ZG72xGLbes9l7okhQ&_render=json&_callback=myCallback

Tech stack: 

Grunt 
Angular JS 
SASS & Compass 



Set-up: 

npm install 
bower install
grunt / grunt serve
